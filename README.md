# Deepfake-Image-Detection-System

The presented deepfake detection system was created using the EfficientNetB4 architecture in the process of transfer learning.
The objective of the model was to determine whether a certain picture was real or produced via AI.
During the training process, real and deepfake images were used as the input data. They were processed using Python and PyTorch libraries.
In particular, the image augmentation was used to facilitate the learning process. Besides that, the system contains a web application built using Streamlit framework.
Thus, it is possible to state that deep learning and computer vision can be effectively used for detecting manipulated digital objects.
